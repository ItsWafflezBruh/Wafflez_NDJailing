# Wafflez_NDJailing
Fivem police jailing script for NDCore

I've had a hard time finding scripts for NDCore, so I made this jailing script. (It's better than the ND one)


About Wafflez_NDJailing
* allows anyone with the allowed job to /jail
* /jail opens Oxlib menu for jailing nearby players
* Shows timer on screen of time left to server using oxlib
* Option to enable freezing players when jailed
* Option to confiscate inventory while jailed

Dependencies
* NDCore
* OxLib
* Ox_Inventory (If using Config.ConfiscateInventory)
